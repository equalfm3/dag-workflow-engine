# Data Directory

This directory stores runtime data generated by the DAG workflow engine:

- **Dead letter queue exports**: JSON files from `DeadLetterQueue.to_json()`
- **Execution logs**: Task execution history and timing data
- **DAG state snapshots**: Serialized DAG state for resume-after-failure

Sample data is generated dynamically by the `__main__` demos in each module.
No static datasets are required.
